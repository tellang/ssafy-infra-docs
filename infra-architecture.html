5. Security Rules

Public Bind 제한

5432/6379/3306/27017/8080의 0.0.0.0 바인딩 금지.

SSH 강화

Password 로그인 비활성화. Root 직접 로그인 차단. Key-only 접근.

Jenkins 접근 제어

Nginx reverse proxy 경유. JENKINS_ALLOW_CIDR로 IP 제한.

Credential 관리

.env, .pem Git 추적 금지. Jenkins credential store 우선.

Tailscale Auth

Auth key 주기적 회전. Host-level VPN으로 운영자 접근 이중화.

Data Isolation

Main/Dev DB, Redis 완전 분리. Cross-environment access ACL 금지.
