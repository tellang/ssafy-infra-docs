Conventions

포트, Docker, 브랜치, 버저닝, 시크릿, 보안 등 팀 전원이 따라야 하는 인프라 컨벤션입니다.

  Rules & Standards

문서 읽는 순서

1. 지금 상태가 궁금하면 먼저 운영 대시보드
2. 연결 구조와 경계는 아키텍처
3. 실제 실행 절차는 운영 가이드
4. 정책 원문은 컨벤션

빠른 시작

상태 갱신: dashboard refresh command 실행
운영 절차: 운영 가이드에서 단계별 작업 수행
구조 점검: main/dev 분리와 내부 포트 경계 확인

대시보드 갱신 명령

pwsh -File scripts/local/refresh-infra-dashboard-data.ps1 -HostAlias ssafy-ec2
